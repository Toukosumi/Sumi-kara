# Sumi-Kara

多功能视频制作软件 - 支持歌词获取、自动打轴、样式生成等模块化功能

## 技术栈

- **前端**: React + TypeScript + Electron
- **后端**: Python FastAPI
- **视频处理**: FFmpeg

## 文档

- [技术架构设计](docs/技术架构设计.md)
- [模块化标准](docs/模块化标准.md)

## 开发环境要求

- Node.js 18+
- Python 3.10+
- FFmpeg 6+
- Git 2+

## 快速开始

```bash
# 安装 Node.js 依赖
npm install

# 安装 Python 依赖
pip install -r requirements.txt

# 启动开发服务器
npm run dev
```
