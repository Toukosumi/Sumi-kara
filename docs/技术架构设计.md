# Sumi-Kara 视频制作软件 - 技术架构设计

## 1. 项目概述

Sumi-Kara 是一款多功能视频制作软件，支持歌词获取、自动打轴、样式生成等模块化功能。

### 1.1 技术栈总览

| 层级 | 技术选型 | 版本 |
|------|---------|------|
| 前端框架 | React + TypeScript | 18.x |
| UI 组件库 | Ant Design / TailwindCSS | 5.x / 3.x |
| 桌面框架 | Electron | 28.x |
| 后端服务 | Python FastAPI | 0.109.x |
| 视频处理 | FFmpeg | 6.x |
| 进程通信 | IPC + WebSocket | - |
| 构建工具 | Vite + electron-builder | 5.x / 24.x |

## 4. 核心模块设计

### 4.1 前端模块 (React + TypeScript)

| 模块 | 职责 |
|------|------|
| `VideoEditor` | 视频编辑主界面 |
| `Timeline` | 时间轴组件 |
| `PreviewPlayer` | 视频预览播放器 |
| `LyricsPanel` | 歌词面板 |
| `StyleConfigurator` | 样式配置器 |

### 4.2 后端模块 (Python FastAPI)

| API 端点 | 功能 |
|---------|------|
| `POST /api/lyrics/search` | 搜索歌词 |
| `POST /api/lyrics/parse` | 解析歌词文件 |
| `POST /api/alignment/auto` | 自动打轴 |
| `POST /api/style/generate` | 生成样式配置 |
| `POST /api/render` | 渲染视频 |

### 4.3 进程通信

```typescript
// 渲染进程 → 主进程
window.electronAPI.send('render:start', { videoPath, config })

// 主进程 → Python 服务
axios.post('http://localhost:8000/api/render', payload)

// Python 服务 → 前端 (进度推送)
websocket.on('progress', (data) => { ... })
```
